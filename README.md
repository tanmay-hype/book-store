# Book Outlet (Django Project)

A simple Django web application that displays books and their details.

## Features

- View all books
- View detailed book information
- Bestseller indicator
- Dynamic URL routing
- Django ORM database integration

## Tech Stack

- Python
- Django
- SQLite
- HTML / Django Templates
- CSS
